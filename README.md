## 🚀 Tecnologias Utilizadas

O portfólio foi construído com foco em performance e uma estética **Dark Tactical**:

- **Frontend:** HTML5, CSS3 (Custom Cyber Theme), JavaScript (jQuery + Typed.js).
- **Ambiente de Dev:** Debian 12 (Bookworm) via VS Code.
- **Libs Externas:** FontAwesome (Icons), OwlCarousel (Exibição de Labs).

---

## 🛠️ O que você vai encontrar aqui?

### 🔴 Red Team & Pentest
- **Op. Luthieria Flamenca:** Write-up completo de um ataque simulado de Spear Phishing com HTML Smuggling e Metasploit.
- **Web App Security:** Testes de vulnerabilidades em aplicações controladas (OWASP Top 10).

### 🔍 OSINT (Open Source Intelligence)
- Metodologias de reconhecimento e enumeração de superfície de ataque.
- Técnicas de *Data Poisoning* e gestão de pegada digital.

### 🔵 Blue Team & Hardening
- Implementação de SIEM (**Wazuh**) via Docker para monitoramento de eventos.
- Hardening de sistemas Linux e gerenciamento de logs.

### 👾 Low-Level & Malware Research
- Estudos em **Assembly x64 (NASM)** para criação de shellcodes.
- Desenvolvimento de ferramentas de automação em **Python** e **C++**.
